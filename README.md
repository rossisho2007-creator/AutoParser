# 📊 SheetGraph

Transform Excel spreadsheets into professional graphs and formatted reports.

## Features
- 📁 Upload Excel (.xlsx, .xls) and CSV files
- 📊 Interactive Plotly visualizations
- 📑 Professional formatted Excel report generation
- 📈 Multiple graph types (bar, line, scatter, histogram)
- 🎨 Clean, modern interface with Tailwind CSS
- 📥 Download template to get started

## Usage
1. Upload your CSV file/Excel file
2. View interactive graphs
3. Download formatted reports

## Tech Stack
- Flask (Python web framework)
- Plotly (interactive graphs)
- Pandas (data processing)
- OpenPyXL (Excel formatting)
- Tailwind CSS (styling)
