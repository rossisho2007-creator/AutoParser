📋 Microsoft Teams List Setup Guide

Step 1: Create Teams List
  1. Open Microsoft Teams
  2. Go to your channel → Click "+" → Select "Lists"
  3. Create a new list with these columns:
ColumnType
NPKText
KPM IDText
Nama LengkapText
Loan AmountCurrency
Monthly InstallmentCurrency
BranchText
StatusChoice (Pending, Approved, Rejected)
Step 2: Connect AutoInput Pro

Use Power Automate to sync:

  1. Go to make.powerautomate.com
  2. Create new flow → "When HTTP request is received"
  3. Add action → "Create item in SharePoint List"
  4. Map the fields from our API to your Teams List
API Endpoint:
GET /api/teams-list - Returns all submissions in Teams-compatible format
Step 3: Add as Teams Tab
  1. In Teams channel → Click "+"
  2. Select "Website"
  3. Paste your AutoInput Pro URL
  4. Name: "AutoInput Pro"

✅ Done! All submissions will appear in your Teams List automatically.
